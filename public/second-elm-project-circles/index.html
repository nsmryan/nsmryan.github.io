Second Elm Project- Circles

2016-10-15

I have another little Elm program up. This one just animates some circles with random sizes and colors. The most interesting thing about it is that it randomly chooses among a group of transformations that it applies to a preset list of base colors.

This means that the colors are always new and always take characteristics of the hard coded palette that I came up with.

You can see it here

The code is on my github repository in the Elm directory as circles.elm
